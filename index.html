Search

Lookup callsigns! QSO uses the callsign-gem to query the Calllook.info API.

$ ./qso search kd0rwr
KD0RWR, TECHNICIAN CLASS
CASEY J WATSON
722 S DEPEW ST
LAKEWOOD, CO 80226-4852

Add

Contacts are stored in a friendly YAML file.

$ ./qso add kd0rwr

List

Quickly see your contacts

$ ./qso list

722 S DEPEW ST
LAKEWOOD, CO 80226-4852
